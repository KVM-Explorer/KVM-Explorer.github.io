语法复习

指针引用​

虽然指针存储的是一个地址，但是不能将该类型变量的值直接赋予该指针。

#include <iostream>
using namespace std;

int main()
{
   int i=0;
    int *p = i;
    std::cout<<p;
   return 0;
}

include头文件​

include头文件仅仅是引入相关的函数定义，因而在实际编译的过程中头文件是不直接参与编译过程，而是在预处理阶段就将相关的头文件在源文件中展开。

我们常见的include语法主要有两种

#include "XXX"
#include <XXX>

第一种主要用于当前的工程项目文件引用，其会优先查找当前工程的项目文件，如果存在则引用。如果不存在，查找引用的库包含路径下是否存在对应的头文件也就是说，下面两种表述的效果是等价的

#include <stdio.h>
#include "stdio.h"

而对于第二种则一般用于引用系统安装的类库和自己编写的类库中的头文件的引用，如当前项目中包含了一个子工程类库，在引用时我们就可以使用#include <library>进行引用。

因而总结下就是，#include "XXX" 用于当前工程文件引用，#include <XXX> 用于库文件引用
