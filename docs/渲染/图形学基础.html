  • Jaggies 锯齿-空间采样

  • Moire 摩尔纹 - 低采样

  • Wagon wheel effect-时间采样

  • More ...

原因在于采样的频率过低导致信号发生了失真

通过先滤波再采样的方式可以一定程度上缓解走样，这也叫做滤波走样（Blurred Aliasing），注意不是反走样。但是反过来无法达到缓解走样的效果

反走样​

  • MSAA

  • FXAA

  • TAA

MSAA 多重采样并没有提升采样率，因为最终像素点数目不变，而是通过相邻的采样进行近似处理（提高采样率最直接的方法是提高屏幕的分辨率）

FXAA采用后处理确定顶点后更换锯齿边界，利用了图像处理方法

超分辨率/超采样技术

  • DLSS

从低分辨率到高分辨率的图像重建，缺少足够的样本，其问题模型和走样类似。而DLSS则是利用深度学习的方式进行推测拟合缺失的信息。

Z-Buffer算法​

虽然在最终的归一化坐标系中，深度按照Z的坐标来看应该为负值（越远越小），但是实际中为了简化运算我们认为距离均为正值

以像素值对应物体深度为计算对象，而不是原来的三角形排序

摆脱了绘制顺序造成的遮挡异常的问题

但是Zbuffer无法解决透明物体的处理

重心坐标法​

不具备投影的不变性，也就是说三维中属性的计算和投影后属性计算存在不一致。因而应该在三维空间中计算属性后再进行投影（eg深度信息）

Shading Franquency​

虽然在从算法的角度来看，相同条件下低精度模型Phone Shading表现更加光滑，而计算开销更大，而Flat Shading 点、面更加明显 计算开销小，但是随着越来越多的高精度模型的使用不同算法其表现差距从屏幕来看去向一致，反而Flat Shading因其较小的计算量更有优势，但是随着模型面数多到一定程度的诸如UE5演示中的超大型模型，单像素下无法展示其涵盖的众多多边形面数，因而在这种情况下Phone Shading反而又是更加高效的方式

Normal​

法线是用于帮助计算模型着色的辅助信息，因其仅和模型相关，而与其他环境无关。且计算量相对较高，因而在实际应用中往往采用法线贴图的方式进行处理，提前计算法线将其烘培成贴图，提高实时速度。

同时法线的计算仅仅表示方向，因而需要将其进行归一化再使用

顶点法线​

利用三角形共用顶点，根据周围的三角形Noraml进行计算平均。同时在此基础上采用不同的三角形面积进行加权平均也是一种常见的算法。

需要注意的是如立方体角点的计算

纹理过小，插值算法处理

  • 最近邻插值算法 Nearest Interpolation Algorithm

  • 双线性插值算法Bilinear Interpolation Algorithm

    • 水平垂直先后顺序计算可以互相推导Todo？

  • 双三次线性插值 Bicubic Interpolation Algorithm

纹理过大，采样处理

  • 超采样 super sampling

    • 实时计算代价高

    • 需求端变更

    • 点查询 Point Query，查询采样点后的计算周围平均值

  • Mipmap

    • 预处理计算，空间换时间

    • 范围查询，Range Query，近似快速

    • 类似图像金字塔log2次采样，但由于规模递减所以实际累计消耗的空间为原始空间的约43\frac{4}{3}34​

    • 采用三线性插值TirLinear Interpolation ALgorithm计算Mipmap层和层之间插值，得到查询结果

      • 底层级双线性插值得到Color1

      • 高层级底层级线性插值得到最终Color

    • Mipmap的缺陷

      • overblur 受限于采样和查询以方格的尺度进行，难以解决不规则的图案

      • 各向异性过滤 部分解决矩形区域查询，空间代价最大为原来三倍，空间换时间

        • X2，X3，XN

      • EWA过滤

纹理应用​

环境贴图（环境光照）​

存在一个既定假设，环境贴图在无限远处

  • 球形环境贴图 Spherical Environment Map

    • 直接展开表面到二维平面，高纬度区域存在扭曲现象

    • 使用Cube Map进行展开， 构建包围Sphere的立方体，并沿球心到表面点的延长线，将贴图信息进行投影

    • 缺点是立方体丧失了球的部分特性，使得光位置方向信息需要从原始球进行计算再进行投影

  • 立方体贴图 Cube Map

凹凸贴图 Bump Mapping​

height Map 影响法线Normal Map进而最终影响Shading

height Map进而达到不改变原有简单模型的情况下呈现复杂的表面效果（Fake，但是在求解Shadow Map的时候会露馅，其阴影呈现出原有几何体形状）

如何计算新的法线​

采用局部坐标系

位移贴图 Displacement Mapping​

根据纹理贴图将实际的三角形进行位移，呈现出真正的不规则形体，在阴影下呈现出正常效果。可以利用几何着色器进行实现，

缺点是要求高度精细的原始模型保证其在位移后保证不会出现过于明显的棱角，能够完整反应纹理情况。但是DirectX提供了动态细分三角形的方法使得，可以使用低精度模型，在需要时对局部三角形进行细分

三维纹理​

三维噪声函数，根据点在三维空间中的位置动态生成其对应贴图

环境光遮蔽​

预计算环境光照信息并生成贴图，在实时中，将环境光贴图和实际模型对应区域着色结果乘积即可得到最终结果

3D Texture and Volume Rendering​

有别于之前的表面着色，在医学方面诸如CT等二维扫描的数据生成三维模型的过程因为其不仅仅局限于表面，需要根据其在三维空间的位置进行生成着色和贴图
