C++开发环境配置

目标​

为实现简单易用统一的C++跨平台开发环境，我们采用了vscode作为我们的集成开发环境，其组成和依赖如下列表所示

  • vscode

  • xmake

  • 包管理工具

    • xmake-repo

    • vcpkg

    • apt

XMake​

Window​

  1. 点击链接Releases · xmake-io/xmake (github.com)根据自己操作系统位数下载xmake-master.win32.exe或xmake-master.win64.exe安装包

  2. 安装完毕后重启电脑使环境变量生效

Ubuntu Linux​

sudo add-apt-repository ppa:xmake-io/xmake
sudo apt update
sudo apt install xmake

Ubuntu Linux​

方法一​

  1. vscode推荐从官网下载deb版本安装包进行安装。

  1. 打开终端，按照如下所示指令进行安装
cd download
sudo dpkg -i <filename>.deb

方法二​

  1. 从官网导入gpa vscode源
sudo apt-get install wget gpg
wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
sudo install -D -o root -g root -m 644 packages.microsoft.gpg /etc/apt/keyrings/packages.microsoft.gpg
sudo sh -c 'echo "deb [arch=amd64,arm64,armhf signed-by=/etc/apt/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
rm -f packages.microsoft.gpg
  1. 打开终端，使用apt进行安装配置
sudo apt install apt-transport-https
sudo apt update
sudo apt install code # or code-insiders

包管理工具​

vcpkg​

安装​

  1. Clone vcpkg源码到本地,默认clone到当前文件夹
git clone https://github.com/Microsoft/vcpkg.git
  1. 运行脚本下载配置Vcpkg二进制文件
.\vcpkg\bootstrap-vcpkg.bat

终端中直接使用vcpkg，需要在环境变量->用户变量->Path中添加vcpkg文件夹

  1. vcpkg支持在 Powershell 和 Bash 中自动补全命令，包名称及选项（可选）
.\vcpkg integrate powershell

卸载​

  1. 移除vcpkg到当前系统的集成
.\vcpkg integrate remove

  1. 删除配置文件和构建文件,删除%localappdata%/vcpkg文件夹

  2. 删除vcpkg文件夹

安装第三方库​

vcpkg install opencv
